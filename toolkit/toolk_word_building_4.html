Word building [4]

Words (lexemes) are built from morphemes - smaller parts with own meaning. Compounding and derivation are word building processes. Inflection grammatically adjusts words.

Introduction

Some words can be divided into separate parts, others cannot.

boomerang (n) → boom rang (changes meaning)
carpet (n) → car pet (changes meaning) 
catnap (n) → cat nap (retains meaning)

Words are built from smaller units called morphemes.

Morphemes

Morphemes are the smallest units of the language. They combine with others to form words. There are two types of morphemes:

Type Description
Free morphemes Can attach to other morphemes or stand alone.

hair, brush → hair-brush

airbrushed → air-brush-ed
Bound morphemes Cannot stand alone.

biology → bio- (makes no sense)

Building processes

A building process results in a new word.

Process Description
Compounding Free morphemes join;

homepage → home-page

fingerprint → finger-print
Derivation Bound morphemes attach to a Stem.

builder → build-er [stem-bound]

disagree → dis-agree [bound-stem]

happy → happy-ly → happi-ly [stem-bound]

Derivation

Bound morphemes attach to a Stem. They are derivational affixes. There are different types of derivational affixes. They are used to create completely new words that have not been seen.
Type Description Examples Maintain word class
Prefixes Word class maintained.

acceptable (adj) → unacceptable (adj)

Exception: friend (n) → befriend (v)

 un-, be-, mis- Yes (generally)
Suffixes Word class typically changes

happy (adj) → happiness (n)

happy (adj) → happily (adv)

Exception: weak (v) → weaken (v)

 -ness, -ly, -ship No (generally)

Words and lexemes

trap*, traps, trapped
entrap*, entrapment*, entrapping

There are 3 distinct words and 6 lexemes*. Words can have different forms, resulting in lexemes.

So, <trap> is a lexeme and <trapped> is a lexeme having <trap> as its base.

The set of all lexemes is called lexis. Inflection forms lexemes.

Inflection

Inflection grammatically adjusts a word. The result is a separate lexeme.

Important: Inflection does NOT change the word class.

There are regular and irregular inflectional affixes. Irregular ones have alternative forms.

Past participle: -be → been, eaten, driven (regular); BUT dealt, understood (irreg.)
Plural form: -s → houses, meals; BUT mouse → mice, fungus → fungi (irreg.)

Allomorphs

Alternative forms not only occur with inflectional morphemes but in general with morphemes.

Examples:

five → fifteen NOT fiveteen (compounding)

leave → left NOT leaved (irregular inflection)

incompetent → impossible NOT inpossible (derivation)
