Word classes [3]

Word classes classify words of sentence structures. All members of a word class behave in the same way. Use the substitution rule to test word class behaviour.

Why word classes?

Word classes replace the <Person/thing - Action - Quality> approach. This approach is no longer acceptable in modern linguistics.

She's got a nasty <cough>.
<Cough> is considered an action, but here used as a noun (not technically a thing).
<Slow> down at the lights.

<Slow> is considered a quality, but here used as a verb.

Meaning does not provide accurate classification as shown. Using word classes is a better approach.

Definition

Word classes classify words by how they behave in sentences. Observing the behaviour of words in sentences lets us deduce their word class memberships. Members of a word class show the same behaviour.

Do you like the <apple/woman/car>? (noun)

All nouns behave the same here.

She is <great/angry/bitchy>? (adjective)

All adjectives behave the same here.

Open classes

Open words classed can take on new members.

Closed classes

Closed word classes cannot (easily) take on new members.

Substitution rule

If words { w1, w2, w3 } belong to the same word class, try to replace them with one another. If the resulting structure remains grammatical, they belong to the same word class.

Multi-membership

She's more than just a <pretty> face.
I'm <pretty> sure they were more than $100.00.

First sentence <pretty> is an adjective. Second sentence <pretty> in a degree adverb (pretty, very, rather, quite: how much?)

It shows that a word can belong to more than one word class.
Examples:
We could <go> to the aquarium (verb)
Is it my <go> next? (noun)
The <last> time I saw her (adjective)
A typical NASA engine will <last> 2.8 years (verb)

Membership transitioning

Some contexts incorrectly assign words to wrong word classes. Over time, the incorrect assignment is accepted.

Examples:

  • Handbag (n) → handbag (v): 'She handbagged him.' (She's domineering)
  • Friend (n) → friend (v), defriend (v), unfriend (v)
  • Action (n) → action (v): 'When is the request going to be actioned?' (Business speak)
