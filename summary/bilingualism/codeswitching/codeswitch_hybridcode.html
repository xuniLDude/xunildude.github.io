Hybrid code or Code mixing

Hybrid code is the result of the convergence of local varieties of two languages.

Hybrid code is the result of the convergence of local varieties of two languages. The two varieties converge phonologically and syntactically into one code called hybrid or mixed code. This happens because codeswitching become habitual in the community.

This generally occurs in outer circle Englishes. Such hybrids are often denigrated and regarded as inappropriate in a public domain, but accepted by locals and used in TV shows or local movies.

Examples:

  • Singlish (Singapore)
  • Manglish (Malaysia)
  • Camfranlais (Cameroon)
  • Hinglish (India)
  • Kitchen language (Cape Town)
